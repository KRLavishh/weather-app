# Weather App

A Python weather app that fetches real-time weather data using the OpenWeatherMap API.

## How to use
- Run weather.py
- Enter a city name
- Get current weather data

## Features
- Fetches live weather data for any city
- Displays temperature, feels like, humidity, and conditions
- Search multiple cities without restarting
- Graceful error handling for invalid cities and no internet connection


## Technologies
- Python
- Requests library
- OpenWeatherMap API
